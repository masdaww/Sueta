# СуетаШоп

Пародийный маркетплейс в стиле OZON. Прототип, всё локально, никаких реальных платежей.

## Возможности

- **Главная** — герой-баннер, категории, рекомендации (популярное / новинки / скидки), поиск.
- **Каталог** — фильтры (поиск, цена, бренд, скидка, наличие), сортировки, breadcrumb, рекомендации.
- **Карточка товара** — галерея, бэйджи, рейтинг, отзывы (с модерацией), добавление в избранное и корзину.
- **Корзина** — изменение количества, удаление, очистка с подтверждением, расчёт скидок.
- **Оформление заказа** — контакты, выбор доставки (4 варианта), оплаты (4 варианта), подтверждение.
- **Заказ** — страница успеха, отслеживание (визуальная шкала статусов, симуляция шагов, отмена).
- **Личный кабинет** — профиль, мои заказы (с отменой), избранное, мои отзывы (статусы модерации).
- **Админка** — дашборд (продажи 7д, топ товаров, статистика), CRUD товаров, CRUD пользователей с блокировкой, модерация отзывов, управление статусами заказов.
- **UX** — адаптивная вёрстка, диалоговые окна, тосты, тултипы, подтверждения опасных действий, sticky-хедер.

Все данные хранятся в `localStorage` (через Zustand `persist`). Сбросить демо-данные можно из админки.

## Стек

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- React Router 6
- Zustand 4 (с middleware `persist`)

## Скрипты

```bash
npm install
npm run dev        # dev-сервер на http://localhost:5173
npm run build      # production-сборка в dist/
npm run preview    # локальный просмотр сборки
npm run lint       # ESLint
npm run typecheck  # TS check
```

## Демо-режим админа

В правом верхнем углу есть кнопка `admin` — она включает админ-режим без логина. По умолчанию пользователь — Тагир Суетной, у которого роль `admin`.

## Структура

```
src/
  components/   # переиспользуемые UI-компоненты (Modal, Toaster, Rating, ProductCard...)
  data/seed.ts  # фейковые данные (товары, отзывы, пользователи, доставки)
  layouts/      # Header, Footer
  lib/          # types и форматтеры
  pages/        # экраны приложения
  store/        # Zustand-сторы (catalog, cart, wishlist, orders, session, notify)
```

## Лицензия

Это пародия для прототипа. Все совпадения с реальными маркетплейсами случайны и забавны.
