# ОЗОР — пародийный маркетплейс

Полностью функциональный фронтенд-прототип маркетплейса в стиле OZON, с пародийной озорной начинкой: енотами, душами в качестве оплаты и курьерами-озорниками.

> Задача: «создать пародию на OZON, выглядящую качественно и функционирующую как прототип». Реальные платежи, БД и доставка не используются — всё имитируется.

## Технологии

- **React 19 + TypeScript + Vite**
- **Tailwind CSS** с собственной палитрой
- **React Router v7** — клиентская маршрутизация
- **Lucide React** — иконки
- **Mock API на localStorage** — данные сохраняются между перезагрузками страницы

## Что внутри

### Покупательская часть
- **Главная** — герой, плитка категорий, хиты, новинки, промо-баннеры
- **Каталог** — поиск, фильтры (категория, цена, рейтинг, бренды, наличие), сортировки, пагинация, режимы «сетка/список»
- **Карточка товара** — описание, рейтинг, отзывы, добавление в корзину/избранное, кнопка «Купить», похожие товары, диалог отзыва
- **Корзина** — изменение количества, удаление, промокоды (`ОЗОР10`, `СУЕТА`, `ЕНОТ2026`), подсказки и подтверждения
- **Оформление заказа** — данные получателя, 4 способа доставки (курьер / самовывоз / дрон / енот) и 4 способа оплаты, итоговый диалог успеха

### Личный кабинет
- Профиль с редактированием имени, телефона, адреса и аватара
- История заказов с прогресс-баром статусов и таймлайном изменений
- Подробная страница заказа с возможностью отмены
- Избранное
- Мои отзывы со статусами модерации

### Админ-панель (`/admin`)
- Дашборд: сводная статистика, графики выручки и регистраций за 7 дней, заказы по статусам, топ товаров и категорий
- Управление товарами: CRUD, выбор эмодзи-«картинок» и градиентных фонов, управление бейджами, скрытие
- Управление пользователями: создание, редактирование, **блокировка**, удаление
- Модерация отзывов: одобрение / отклонение / удаление с фильтром по статусу

### UI-инфраструктура
- Полностью адаптивный дизайн (мобильное меню, нижние шторки модалок)
- Аккуратные модальные окна (ESC, клик по подложке, lock body)
- Тосты-уведомления с авто-закрытием
- Подтверждающие диалоги для деструктивных действий
- Скелетоны и empty states с пародийным юмором

## Запуск

```bash
npm install
npm run dev    # http://localhost:5173
```

Для production-сборки:

```bash
npm run build
npm run preview
```

Проверки:

```bash
npm run lint
```

## Демо-аккаунты

| Email             | Пароль       | Роль          |
|-------------------|--------------|----------------|
| `user@ozor.ru`    | `user12345`  | Покупатель     |
| `admin@ozor.ru`   | `admin12345` | Администратор  |

При первом запуске генерируется пачка демо-заказов, чтобы админка сразу была живой. Сбросить состояние можно очистив localStorage (`ozor.*`).

## Промокоды

- `ОЗОР10` — −10%
- `СУЕТА` — −15%
- `ЕНОТ2026` — −25%

## Заметка о пародии

Всё, что выглядит как товар или категория, — выдумка ради улыбки. Никаких настоящих платежей, доставок, енотов и душ. Если в реальной жизни что-то совпало — это совпадение, и енот тут ни при чём.
